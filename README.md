ここには、研究や課題で作成したプログラムを置いてあります。

othello.cpp:研究室のチュートリアル課題で作ったプログラム。αβ法+反復深化で一定時間以内に着手します。同枠のparameter.txtから評価関数を読み込んで動作。

tagging.cc:卒論で実装した、ビームサーチを用いた品詞タグ（POSタグ）付けプログラム。PTBコーパスの形式で学習・タグ付けを行う。

parsing.cc:同じく卒論で実装した、最小有向全域木を用いた係り受け解析プログラム。CoNLL08 Shared Taskの形式で学習・解析を行う。

tagging2.cc:上記の品詞タグ付けに上記の係り受け解析を援用した、卒論で最終的に使用したプログラム。
